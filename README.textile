h1. About

This is 101ideas.cz:"http://101ideas.cz". It is written in "Ace":https://github.com/botanicus/ace, a powerful generator of static sites.

h1. Usage

_*NOTE:* You need Ruby 1.9.2 or later._

# Install all the dependencies via bundler: @bundle install@.
# Generate the output by running @bundle exec ./tasks.rb generate@.
# Run @bundle exec ./config.ru@ and view the content at "localhost:9292":http://localhost:9292/.
